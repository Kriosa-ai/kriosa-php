# Gateman PHP - AI-Enhanced Security Middleware

[![PHP Version](https://img.shields.io/badge/PHP-8.0%2B-blue.svg)](https://packagist.org/packages/gateman/ai-middleware)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Gateman is an intelligent security middleware for PHP applications that acts as a smart layer between users and your application, detecting, filtering, and logging malicious activities before they reach your core application.

## Features

- 🛡️ **AI-Powered Threat Detection** - Advanced heuristic analysis with pattern recognition
- 🔍 **Multiple Attack Detection** - SQL Injection, XSS, Path Traversal, Rate Limiting
- 🚀 **Framework Agnostic** - Works with Laravel, Symfony, and vanilla PHP
- 📊 **Real-time Analytics** - Threat scoring and confidence levels
- ⚡ **High Performance** - Minimal overhead with maximum protection
- 📝 **Comprehensive Logging** - Detailed security event logging

## Installation

```bash
composer require gateman/ai-middleware
